A set of experiments have been completed where gel, media composition, flow shear and centrifuge regimen have been optimised to improve bone cells (osteoblast) growth within a microfluidic hydrogelled bone-on-chip. The dataset that contains measurements of just excitation / lower light wavelengths from fluorescence in assay plate readers is  the ML_FlowCentrifuge.csv  file. The measurements that contain both excitation / lower light wavelenghts and emission / upper wavelenghts from the assay plate readers are contained in the ML_FlowCentrifuge_all.csv file. These can be used to train and compare future ML/AI predictive models.      These datasets superseded a similar set of datasets, a ML_FlowCentrifuge_init_ML.csv file derived from initial (pre-stabilised assay measurements) was a noisier dataset used to test a set of ML predictive regression models. An extended version of this dataset called A ML_FlowCentrifuge_all_init_ML.csv using additional wavelength measurements was used to train and benchmark a set of AI and deep learning predictive regression models. The latter two datasets may be used to train future models on noisier pre-processed data but understand that these were pre-finalised or stabilised assay values.
